# Overview

NeuroSense is a low-cost wearable neuromotor assessment platform designed to measure and visualize hand movement using inertial sensors and modern web technologies.

The project combines principles from:

- Physics
- Biomedical Engineering
- Human Biomechanics
- Embedded Systems
- Signal Processing
- Web Development

The objective is to investigate whether affordable consumer-grade IMUs can provide meaningful quantitative measurements of hand motion for research and educational purposes.

# Research Questions

The current research focuses on four primary questions.

1. Can low-cost IMUs quantify hand movement?

2. Can wearable sensors objectively measure movement stability?

3. Can browser-based dashboards improve interpretation of motion data?

4. How can affordable biomedical technology support research accessibility?
